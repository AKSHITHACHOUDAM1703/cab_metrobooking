# Metro + Cab Booking

A simple Streamlit application for booking metro tickets and optionally adding a cab ride to the journey.

## Features

- Select the starting and destination metro stations
- Choose the number of metro tickets
- Add an optional cab ride
- View a fare breakdown and booking confirmation

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/AKSHITHACHOUDAM1703/cab_metrobooking.git
   cd cab_metrobooking
   ```

2. Install Streamlit:

   ```bash
   python -m pip install streamlit
   ```

## Run the application

```bash
python -m streamlit run cab.py
```

Streamlit will display a local URL in the terminal. Open that URL in a browser to use the application.
